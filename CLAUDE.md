# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project

carSpot is a car spotting web application. The project is in early development — no framework or build tooling has been chosen yet.

## Git Workflow

This project uses GitHub at https://github.com/Josephadam/carSpot.

After every meaningful change:
1. Stage specific files (never `git add -A` blindly)
2. Commit with a clear, descriptive message
3. Push to `origin main`

The user expects all changes to be committed and pushed so there is always a recoverable history.
